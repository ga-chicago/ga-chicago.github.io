Subjects Object

Using this simple HTML form, we can add and remove topics from our javascript object called 'subjects'.

Either using the form, or using JavaScript functions, we can add, remove, and list all topics available in our object's topics list.

To add a topic in JS, use the syntax subjects.addTopic('your topic here');

To remove a topic in JS, use the syntax subjects.removeTopic('topic to remove here');

To list the current available topics, use the syntax subjects.listTopics();
